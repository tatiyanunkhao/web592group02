หนังสือคู่มือช่างคอม 2016 ฉบับสมบูรณ์

หนังสือคู่มือช่างคอม 2016 ฉบับสมบูรณ์ เป็นหนังสือที่มีเนื้อหาข้อมูลอัดแน่นทั้งที่เป็นทฤษฎีและปฏิบัติ สำหรับทั้งคอมพิวเตอร์พีซีและโน้ตบุ๊ค ในส่วนของทฤษฎี จะพูดถึงรายละเอียดและเทคโนโลยีของชิ้นส่วนต่างๆ ตลอดจนวิธีการจัดสเป็คและเลือกซื้อชิ้นส่วนต่างๆ เพื่อนำมาประกอบเป็นเครื่องพีซีด้วยตัวเอง โดยมีเนื้อหาละเอียด ครบถ้วน ทันสมัย และให้ข้อมูลที่ถูกต้องอัพเดตล่าสุด ในส่วนของปฏิบัติ จะพูดถึงการลงมือประกอบเครื่องคอมพิวเตอร์ด้วยตนเอง เริ่มตั้งแต่จัดเตรียมอุปกรณ์ ลงมือประกอบจนเสร็จ ตั้งค่าไบออส จัดเตรียมฮาร์ดดิสก์ ติดตั้ง Windows 7, 8.1 และ 10 ติดตั้งไดรเวอร์และโปรแกรมใช้งานต่างๆที่จำเป็น ตลอดจนการตรวจสอบ ดูแล และแก้ไขปัญหาคอมพิวเตอร์ด้วยตนเอง

คู่มือสร้างเว็บไซต์ ฉบับสมบูรณ์

หนังสือ “คู่มือสร้างเว็บไซต์ ฉบับสมบูรณ์ ” ที่จะช่วยให้คุณสร้างเว็บไซต์ได้อย่างมืออาชีพ เริ่มตั้งแต่การออกแบบ วางแผน เตรียมรูปภาพด้วย Photoshop ไม่ว่าจะเป็นออกแบบหน้าเว็บ, สร้างปุ่ม, ป้ายโฆษณา และภาพแอนิเมชั่น จัดหน้าเว็บเพจด้วย Dreamweaver ด้วยเทคนิคต่างๆ, สร้างมูฟฟวี่ด้วย Flash รวมถึงการจองโดเมนเนมและวางเว็บโฮสติ้งขึ้นสู่เครือข่ายอินเทอร์เน็ต และเรื่องอื่นๆที่เกี่ยวข้อง ครบถ้วนในเล่มเดียว

หนังสือคู่มือสร้าง ebook ด้วย iBooks Author

หนังสือคู่มือสร้าง ebook ด้วย iBooks Author เนื้อหาในเล่ม เหมาะสำหรับผู้ที่มีความสนใจเกี่ยวกับการสร้าง ebook ซึ่งได้อธิบายถึงหลักการพื้นฐานของการสร้างหนังสือดิจิตอล วิธีใช้เครื่องมือและขั้นตอนต่างๆ จนได้ไฟล์หนังสือ 1 เล่ม ที่สามารถเปิดอ่านและเผยแพร่ต่อได้ด้วยเนื้อหาครบถ้วน กระชับ เข้าใจง่าย มีเนื้อหาครอบคลุมโดยละเอียด จึงเหมาะสำหรับผู้เริ่มต้นแม้จะไม่เคยใช้งาน iBook Author มาก่อนและผู้ที่สนใจทั่วไป สามารถสั่งซื้อ หนังสือคู่มือสร้าง ebook ด้วย iBooks Author

Insight PHP ฉบับสมบูรณ์

  • - เรียนรู้การเขียนโปรแกรม PHP ตั้งแต่ขั้นพื้นฐานไปจนถึงการประยุกต์ใช้งาน
  • - ขั้นตอนการติดตั้งเว็บเซิร์ฟเวอร์และ PHP ทั้งบน Windows และ Linux
  • - การใช้งานระบบฐานข้อมูล MySQL และการเขียนโปรแกรม PHP เพื่อติดต่อกับฐานข้อมูล MySQL และ Microsoft Access ผ่าน ODBC
  • - แก้ปัญหาการเปลี่ยนระบบฐานข้อมูลด้วย PDO
  • - PHP กับการเขียนโปรแกรมเชิงวัตถุ (OOP)
  • - อธิบายการใช้เทคนิค AJAX ร่วมกับ PHP อย่างละเอียด พร้อมตัวอย่างการประยุกต์ใช้
  • - กรณีศึกษา (case study) ที่นำไปใช้งานได้จริง RemoteDrive 1.1 (ปรับปรุงใหม่), Web Mail, Webboard และ Counter

    • คู่มือเขียนโปรแกรมด้วย ภาษา C ฉบับสมบูรณ์

    หนังสือคู่มือเขียนโปรแกรมด้วยภาษา C เล่มนี้ เป็นฉบับปรับปรุงใหม่ล่าสุด พิมพ์สองสีตลอดทั้งเล่ม ที่ใช้ได้ผลดีในการเรียนการสอนตามสถาบันต่างๆ และเป็นที่ยอมรับมานานกว่าสิบปี ว่าอ่านเข้าใจง่ายที่สุด พิมพ์จำหน่ายไปแล้วหลายหมื่นเล่ม ด้วยวิธีอธิบายและภาพประกอบที่เห็นขั้นตอนต่างๆ อย่างชัดเจน สามารถทำตามได้จริง จึงใช้ได้สำหรับการเรียนการสอนในชั้นเรียน หรือจะเป็นคู่มือเพื่อศึกษาด้วยตนเองก็ได้ สามารถใช้อ่านได้ตั้งแต่ผู้ที่เริ่มต้นศึกษา จนถึงระดับสูงสุด ซื้อหนังสือคู่มือเขียนโปรแกรมด้วยภาษา C ฉบับปรับปรุงล่าสุด

    สร้าง Responsive Learning ด้วย Adobe Captivate 8

    หนึ่งเล่มที่น่าสนใจเป็นอย่างมาก หนังสือที่สอนการสร้างสื่อมัลติมีเดียด้วยโปรแกรม Adobe Captivate 8 หนังสือสร้าง Responsive Learning ด้วย Adobe Captivate 8 เนื้อหาของหนังสือเล่มนี้เป็นการอธิบายการใช้งาน Adobe Captivate 8 ไว้ได้อย่างละเอียดมาก ผู้เขียนลําดับเนื้อหาเป็นขั้นตอนให้สามารถศึกษาเรียนรู้ได้จากระดับเริ่มต้นไปยังระดับสูง เหมาะสําหรับนักเรียน นักศึกษา ครู-อาจารย์ นักพัฒนาสื่อ และผู้สนใจทั่วไป ผู้เขียนหวังว่าหนังสือเล่มนี้จะสามารถจุดประกายความคิดสร้างสรรค์ในการสร้างสื่อการเรียนรู้ได้เป็นอย่างดี

    หนังสือคู่มือเขียนแอพ Android ด้วย Android Studio

    ปัจจุบันไม่ว่าใคร ๆ คงปฎิเสธไม่ได้ว่าคนส่วนใหญ่ใช้เวลาอยู่กับโทรศัพท์สมาร์ทโฟน อาชีพที่ตามมาที่ใครหลาย ๆ อยากทำคือการสร้างแอพพลิเคชั่น หนังสือคู่มือเขียนแอพ Android ด้วย Android Studio เป็นคู่มือที่ช่วยสอนการเขียนแอพ Android โดยการใช้ Android Studio ทำให้คุณได้เรียนรู้ตั้งแต่การติดตั้งซอฟต์แวร์ต่างๆที่จำเป็น ไปจนถึงเมื่อคุณสร้างแอพและนำแอพเข้าสู่ Google Play อธิบายเนื้อหาอย่างเป็นลำดับขั้นตอน เพื่อให้คุณมีพื้นฐานที่แข็งแรงสำหรับการก้าวไปเป็นนักพัฒนามืออาชีพต่อไปเพื่อให้แอพของคุณ “โกอินเตอร์” เต็มตัว เนื้อหาครอบคลุมเรื่อง Graphics, Multimedia, File I/O, SQLite Database, Content Provider, Multi-touch, Location, Maps, Sensor, Widget, Live Wallpaper, Activity, Intent, Service, Broadcast Receiver และ Facebook

    หนังสือคู่มือ Mac OS X EI Capitan ฉบับสมบูรณ์

    หนังสือคู่มือ Mac OS X EI Capitan ฉบับสมบูรณ์ หนังสือเล่มนี้จะช่วยให้คุณใช้งาน Mac ได้ง่ายขึ้น โดยแสดงขั้นตอนการใช้งานตั้งแต่เริ่มเปิด/ปิดเครื่อง การใช้งานแอพพื้นฐาน รวมไปถึงแอพที่แจกให้ดาวน์โหลดฟรีสำหรับผู้ที่ซื้อเครื่องใหม่อย่าง Pages,Numbers,Keynote, iMovie และ Garage Band ด้วย ทั้งนี้คุณสามารถเชื่อมโยงข้อมูลในเครื่องไปยังอุปกรณ์ iDevices ต่างๆ เช่น iPad, iPhone เพื่อไม่ให้พลาดข้อมูลสำคัญ ผ่าน iCloud ได้ นอกจากนี้ยังแนะนำวิธีการอัพเดทและติดตั้ง OS X El Capitan, ติดตั้ง Windows บนเครื่องMac, ค้นหาเครื่อง Mac เมื่อทำหาย, การสำรอง/กู้คืนข้อมูล, การฟอร์แมตและแบ่งพาร์ทิชั่น รวมไปถึงการดูแลรักษาเครื่องอีกด้วย

    สร้าง Windows Application ด้วย Visual Basic 2015

    Visual Basic นั้นเป็นภาษาที่ผ่านการพัฒนามาอย่างต่อเนื่อง จนมาถึงรุ่นล่าสุดคือเวอร์ชั่น 2015 ซึ่งได้มีการปรับปรุงประสิทธิภาพให้สามารถประมวลผลได้เร็วขึ้นกว่าเดิม และเพิ่มเติมความสามารถใหม่ๆ เข้ามาหลายอย่าง เช่น String Interpolation, Multiline String เป็นต้น ซึ่งเราสามารถนำไปประยุกต์ใช้กับการเขียนโปรแกรมทั่วๆ ไปได้เป็นอย่างดี แต่อย่างไรก็ตามถึงแม้สิ่งที่มีใน Visual Basic จะเพียบพร้อม แต่คงจะไม่มีประโยชน์อันใดเลย หากเราขาดความรู้และความเข้าใจการนำไปประยุกต์ใช้งาน ด้วยเหตุนี้ผู้เขียนจึงได้จัดทำหนังสือเล่มนี้ขึ้น เพื่อรวบรวมความรู้ในภาษา Visual Basic ที่มีอยู่ทั้งหมดจากประสบการณ์ที่เคยใช้งานมาหลายเวอร์ชั่น โดยมีตัวอย่างและแบบฝึกหัดท้ายบทเพื่อเสริมทักษะ พร้อมเทคนิคการนำไปประยุกต์ใช้งานจากเวิร์คช็อปที่หลากหลาย มุ่งเน้นให้ผู้อ่านได้เข้าใจหลักการสำคัญต่างๆ ที่มีใน Visual Basic ควบคู่ไปกับแนวทางการนำไปประยุกต์ใช้งานเพื่อสร้างแอปพลิเคชั่นที่สมบูรณ์แบบ

    ระบบฐานข้อมูล (Database Systems) ฉบับปรับปรุงเพิ่มเติม

    "ฐานข้อมูล" จัดเป็นทรัพยากรอันทรงค่ายิ่งสำหรับทุกองค์กร ดังจะเห็นได้จากระบบสารสนเทศที่พัฒนาขึ้นมานั้น ล้วนเกี่ยวข้องกับเทคโนโลยีฐานข้อมูลที่นำมาใช้จัดเก็บข้อมูลแทบทั้งสิ้นโดยเฉพาะฐานข้อมูลเชิงสัมพันธ์ (Relational Database) อย่างไรก็ตาม ก็มีนักคอมพิวเตอร์อยู่จำนวนไม่น้อยที่มีความรู้ด้านการเขียนโปรแกรมที่ดี แต่ยังขาดทักษะความรู้ด้านการออกแบบฐานข้อมูล กล่าวคือ ไม่รู้ว่าจะเขียนโปรแกรมเพื่อจัดเก็บข้อมูลอะไรบ้าง หรือข้อมูลต่างๆ ต้องมีการเชื่อมโยงกันอย่างไร สิ่งเหล่านี้ถือเป็นความท้าทายของนักคอมพิวเตอร์รุ่นใหม่ที่จำเป็นต้องเรียนรู้เพิ่มเติมเพื่อยกระดับความสามารถของตนให้สูงยิ่งขึ้น อีกทั้งยังเป็นการสร้างโอกาสความก้าวหน้าในงานได้อีกทางหนึ่ง ดังนั้น หนังสือ ระบบฐานข้อมูล เล่มนี้จะช่วยเติมเต็มให้คุณได้เรียนรู้ถึงเทคนิคการออกแบบฐานข้อมูลที่ถูกต้องตามหลักวิชา เพื่อเสริมสร้างความแข็งแกร่งและพร้อมที่จะยืนหยัดในวิชาชีพได้อย่างเต็มภาคภูมิโดยไม่ยาก "ระบบฐานข้อมูล (Database Systems) ฉบับปรับปรุงเพิ่มเติม" เล่มนี้ ได้รับการออกแบบเนื้อหาให้ครอบคลุมภาคทฤษฎีและภาคปฏิบัติ โดยได้ถ่ายทอดความรู้ไว้อย่างเพียงพอต่อการศึกษาในระดับอุดมศึกษา จึงเหมาะสำหรับใช้เป็นหนังสือประกอบการเรียนหรืออ้างอิงของคณาจารย์ นิสิตนักศึกษาที่ศึกษาอยู่ในแขนงวิชาทางคอมพิวเตอร์และไอที รวมถึงผู้สนใจทั่วไปที่ต้องการยกระดับความรู้ด้านระบบฐานข้อมูล

    พัฒนาเว็บแอปพลิเคชั่นด้วย JavaScript

    เมื่อกลางเดือนมิถุนายน 2558 ภาษาจาวาสคริปต์ได้ออกมาตรฐานตัวใหม่ชื่อ ECMAScript 6 (หรือ ES6) ด้วยเหตุนี้หนังสือเล่มนี้จึงเกิดขึ้นมา เพื่อเป็นคู่มือให้คนไทยได้อ่าน และปูพื้นฐานการพัฒนาจาวาสคริปต์ตัวใหม่โดยเฉพาะ จะได้ตามทันเทคโนโลยียุคใหม่ที่ก้าวเดินอย่างรวดเร็วปานติดจรวด โดยเนื้อหาในเล่มจะโฟกัสเกี่ยวกับจาวาสคริปต์อย่างเดียวล้วนๆ ไม่เหมือนกับหนังสือเล่มอื่นที่มีอยู่ในท้องตลาด ซึ่งมักจะรวมภาษา HTML กับ CSS แทรกเข้ามาอยู่ในเนื้อหาเดียวกัน สำหรับสาเหตุที่แยกเขียนออกมาอีกเล่ม เพราะฟีเจอร์ที่เพิ่มเข้ามาในมาตรฐานตัวใหม่ค่อนข้างเยอะมาก และมีหน้าตาต่างจากของเก่าพอสมควร เมื่อคุณอ่านเล่มนี้จบ จะรู้ทันทีว่าจาวาสคริปต์ที่คุณเคยรู้จัก มันนับเวลาถอยหลัง ใกล้หมดอายุลงแล้วจริงๆ หนังสือเล่มนี้จะประโยชน์ต่อผู้อ่านทุกท่าน ที่สนใจศาสตร์ด้านคอมพิวเตอร์ ไม่ใช่แค่เพียงนักพัฒนาซอฟแวร์อย่างเดียว เพราะในอนาคตการเขียนโปรแกรมยุคใหม่ จะหลีกเลี่ยงจาวาสคริปต์ตัวใหม่นี้ไม่ได้อย่างเด็ดขาด

    AutoCAD 2016 Complete Guide 2D/3D

    AutoCAD เป็นโปรแกรมที่ครอบคลุมทุกส่วนกระบวนการของงานเขียนแบบและออกแบบด้วยคอมพิวเตอร์ ไม่ว่าจะเป็นงานเขียนแบบ 2 มิติหรือ 3 มิติ หนังสือ "AutoCAD 2016 Complete Guide 2D/3D" เล่มนี้ มีเนื้อหาครบถ้วน ครอบคลุมทั้งงานเขียนแบบ 2 มิติ และ 3มิติ ด้วยวิธีการสอนแบบเข้าใจง่าย ทำตามได้แม้เป็นมือใหม่ และใช้งานได้ครบทุกฟังก์ชั่นของโปรแกรม จากรายละเอียดการกำหนดค่าครบทุกคำสั่ง เหมาะสำหรับผู้ที่ต้องการเรียนรู้ด้วยตนเอง พร้อม Workshop การทำงานจริงระดับมืออาชีพ
